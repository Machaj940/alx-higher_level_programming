===============================
 How to Use 1-my_list.py
===============================

Import module:
       >>> MyList = __import__('1-my_list').MyList

Check regular sorted list:
      >>> my_list = MyList()
      >>> my_list.append(1)
      >>> my_list.append(4)
      >>> my_list.append(2)
      >>> my_list.append(3)
      >>> my_list.append(5)
      >>> my_list.print_sorted()
      [1, 2, 3, 4, 5]

Check sorted list with negatives:
      >>> my_list = MyList()
      >>> my_list.append(-1)
      >>> my_list.append(-4)
      >>> my_list.append(-2)
      >>> my_list.append(-3)
      >>> my_list.append(-5)
      >>> my_list.print_sorted()
      [-5, -4, -3, -2, -1]

Check sorted list with negatives and positives:
      >>> my_list = MyList()
      >>> my_list.append(-10)
      >>> my_list.append(-40)
      >>> my_list.append(-20)
      >>> my_list.append(30)
      >>> my_list.append(50)
      >>> my_list.print_sorted()
      [-40, -20, -10, 30, 50]
